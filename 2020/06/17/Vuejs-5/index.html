[Vue.js 筆記 (五) Vue.js常用API]

發表於 2020-06-17 | 分類於 Vue.js筆記 |

set( )

常常會無法預先把資料結構定義的非常的完整，尤其是 AJAX 所找回的資料無法預先定義，就可以使用 set() 寫入資料

Extend( )

可以將多個元件或者對象內相同的內容集中至 Extend() 內作為基底，之後需要用到其內容時，可以直接調出使用

  • 當我使用在 childTwo 使用 extend() 之後，一樣可以新增各個物件或是更改文字等，這樣也都不會變動到原本 extend() 的內容，並且直接在 childTwo 進行覆蓋內容。

filter( )

自訂畫面資料呈現格式

mixin( )

mixin() 簡單來說一些常用的組件可以放入 mixin() 裡方便調用，也不需要用到相同組件時重複編寫，混入多個元件也能進行全域註冊，在需要該功能時再「混入」，而且不會使類的關係變得複雜，並且 mixin() 對象可以包含任何組件選項(生命週期、data、computed、methods…等)

mixin( ) vs. extend( )

  • mixin()能夠加入多個組件，對象可以包含任何組件選項(生命週期、data、computed、methods…等)，但必須注意的mixin()是在存在於全域註冊，它會影響每個單獨創建的 Vue 實例。

  • extend()是只能擴展一個元件去做延伸，而且必須是函數。

自定義指令(directives)

  • 鉤子函數：鉤子（Hook）是 Windows 消息處理機制的一個很重要的内容，應用程序可以通過鉤子機制截獲處理 Window 消息或是其他一些特定事件。
