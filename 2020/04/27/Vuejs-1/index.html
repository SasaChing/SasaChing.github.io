Vue.js 筆記 (一) 安裝＆起手式

Vue.js 載入

  • 使用 CDN 或官網下載 Vue.js 以<script>載入HTML內。

  • NPM

    - CDN 或官網下載 Vue.js

    版本

  • 開發版本(Development version)：完整的提示警告

  • 生產版本(Production version)：壓縮版本，無提示警告

開發時建議使用開發版本(Development version)

- NPM

$ npm install vue

- Vue Devtools 套件

  • Chrome 開發者工具的 Vue Devtools 套件，能夠方便審查和測試應用。

Vue.js 起手式

宣告

  • Vue.js 建議使用id來做綁定，也可以用Class。但為何建議用id綁定是因為 Vue 一次只能綁定一個元素，而id也是獨一無二的，所以使用id綁定比較好。

  • 通常一個頁面只會建立一個 Vue ，但建立兩個也是可以的，並且 Vue 是不能建立巢狀。

定義數據對象

  • 兩個大括號用於輸出對象屬性或是函數回傳值

  • el參數是 DOM 元素中的id，意味著el指向#app則變動內容只會在#app內，div外部不受影響

  • data 用於定義屬性，以上範例中有兩個屬性分别為：text、url

  • methods 用於定義的函数，可以通過 return 來回傳函數值（裡面一定是 function ）
