安裝＆載入

Vue.js 載入

  • 使用 CDN 或官網下載 Vue.js 以<script>載入HTML內。
  • NPM

CDN 或官網下載 Vue.js

版本

  • 開發版本(Development version)：完整的提示警告

  • 生產版本(Production version)：壓縮版本，無提示警告

開發時建議使用開發版本(Development version)

NPM

$ npm install vue

Vue Devtools 套件

  • Chrome 開發者工具的 Vue Devtools 套件，能夠方便審查和測試應用。

Vue.js 起手式

宣告

  • Vue.js 建議使用id來做綁定，也可以用Class。但為何建議用id綁定是因為 Vue 一次只能綁定一個元素，而id也是獨一無二的，所以使用id綁定比較好。

  • 通常一個頁面只會建立一個 Vue ，但建立兩個也是可以的，並且 Vue 是不能建立巢狀。

定義數據對象

  • 雙括號用於輸出對象屬性或是函數回傳值

  • el參數是 DOM 元素中的id，意味著el指向#app則變動內容只會在#app內，div外部不受影響

  • data 用於定義屬性，以上範例中有兩個屬性分别為：text、url

  • methods 用於定義的函数，可以通過 return 來回傳函數值（裡面一定是 function ）

Vue.js 概述

資料串接＆綁定

  1. 雙花括號與資料串接

    輸出對象屬性或是函數回傳值

  2. 新增 HTML 屬性

    v-html 可用來直接新增html標籤，且執行標籤功能

  3. 雙向綁定

    v-model 資料動態同步調整，以範例來說，可以使用input變更message內容。在 input、select、textarea、checkbox、radio 等表單元素上建立雙向數據綁定，根據表單上的值，自動更新綁定元素的值。

  4. 輸出文字

    v-text 直接輸出文字資料

v-bind 屬性綁定

使用v-bind綁定html屬性

範例中使用v-bind綁定src屬性來增加圖片網址，並且加入一個class來設置響應式圖片。img-fluid 是 Bootstrap 4 用法。

  • 簡化寫法把v-bind 變換成 : 冒號即可。

模板語法 v-if & v-for

v-for

v-for 綁定陣列到一個自定名稱來渲染

  • item = 自定義名稱 ( 把list的陣列存在裡面 )
  • index = 數字索引值

v-if

v-if 可用來加入判斷式做過濾條件，範例則用年齡小於 25 歲來做篩選。

v-if根據表達式data內的值 ( true 或 false ) 來決定是否插入元素

v-on 處理互動式行為

  • 用於監聽 DOM 事件
  • 添加修飾符：使用修飾符時，順序很重要；相應的代碼會以同樣的順序產生

範例 - 反轉字串

範例 - 修飾符

  • prevent 阻止元素發生默認的行為
  • keyup.enter (按鍵修飾符)增加 Enter 事件
  • 簡化寫法把v-on 變換成@冒號即可。
  • 更多修飾符-官方文件

Class 切換

  • v-bind動態增加class="{ class名稱：判斷式 }"
  • v-on增加click事件，點擊後讓isTransform反轉true / false值

點擊前（false） 點擊後（true）

computed 計算屬性

可使用 computed 取代原有的表達式

表單雙向綁定

預先定義資料結構

  • Vue.js 傾向預先定義資料結構，必須要使用指令前能夠先確認資料結構較好。但也有時不能預先知道資料結構等，所以之後會再介紹如何解決。

元件化

  • 舉例：製作了計數器，但是次數會同步

  • 元件化能夠讓元件獨立作業。
