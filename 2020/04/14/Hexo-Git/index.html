Hexo + Git 部落格開張

初心者寫手，紀錄 Hexo 部落格建立過程和使用問題。

安裝

必須先確認安裝前電腦是否具備以下軟體

  • Node.js
  • Git

若尚未安裝，是不能接續安裝Hexo，所以必須先前處理。

安裝 Hexo

透過 npm 即可輕鬆安裝 Hexo

安裝完後，以下指令能查看安裝的版本

安裝 Hexo Git

若要上傳至 GitHub 必須安裝 Hexo Git ，剛開始安裝時忘記裝這一個，結果摸了很久才發現，
真是浪費我時間啊～但是不熟悉的東西就是會需要摸索才會印象深刻。

建立

安裝 Hexo 後就能開始建立資料夾和檔案

配置上 GitHub

  1. 至 GitHub 上建立一個新的數據庫（Repository）


  2. Repository name 設定為：（你的github帳號）.github.io
    ex : Huangchiching.github.io

  3. 前往 Blog 資料夾內的 _config.yml 檔案

  4. 開啟 _config.yml 後至最底部會看到

  5. 接著修改成

  6. 部署上 GitHub

  7. 完成後就能到 GitHub 查看是否部署成功囉！

配置檔 _config.yml

_config.yml 是 Hexo 的配置設定檔，可以自行設定。

  • 關於 URL（以我本身為範例）
  1. url（網址）的部分是由 GitHub pagesg 分配的網址
  2. root（根目錄）是 Repository 的名稱
  • 會特別提出來的原因是我的 GitHub pagesg 都吃不到樣式，最後才發現我的 root 設置錯誤。

網頁主題

我的 Blog 是使用 Next.Pisces ， 官方文件 有詳細的中文教學，非常簡單。若想要設置別的主題，
可以到 主題區 挑選，決定後下載主題至 themes 資料夾內

下載至 themes 資料夾後，themes/_config.yml (主題設定檔)，
找到以下部分填寫網站主題名稱

註：因為我使用 Next.Pisces ，所以我填寫 next

刪除文章

開始設定

新增標籤頁面

輸入新增標籤頁面指令

建立之後會在 Blog/source/tags 資料夾裡看到 index.md ，並且開啟檔案之後編寫 type 屬性為 tags ，此處的 comments 是可以設定評論功能是否開啟（預設是開啟）

接著打開themes/_config.yml (主題設定檔)，搜尋menu找到以下內容，
並且把你需要的頁面的#移除

新增分類頁面

輸入新增分類頁面指令

建立之後會在 Blog/source/categories 資料夾裡看到 index.md ，並且開啟檔案之後編寫 type 屬性為 categories，建議加上 comments: false 關閉評論功能

接著前往themes/_config.yml (主題設定檔)，修改menu，並且把你需要的頁面的#移除

讓文章加入標籤和分類

以我這個文章做範例，在我的文章Blog/source/_post/Hexo-Git.md文件裡新增updated、categories、tags

updated 更新時間
categories 分類：直接加上分類名稱
tags 標籤：直接加上標籤名稱

繼續閱讀

若文章內容太長，也可以在文章內文裡加入<!-- more -->進行截斷。

加入大頭貼

請在Blog/source新增images資料夾，並且放入圖像後，
前往themes/_config.yml 尋找avatar加入圖像路徑

第三方插件 - 留言板

  1. 請先至 Disqus 創建帳號

  2. 點選下方的 I want to install Disqus on my site

  3. 點選 Mange

  4. 點選 Edit Settings

  5. 填寫 Website URL 並複製你的 Shortname

  6. 回到themes/_config.yml (主題設定檔)中找到disqus，並把剛剛複製好的貼在shortname後面
